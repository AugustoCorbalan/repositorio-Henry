Variables: Son elementos que podemos crear asignandole un nombre que debe cumplir determinadas reglas, que nos permite guardar en memoria un dato. Ya sea un número, string, valor booleano, none, undefined. En JavaScript hay 3 tipos de variables (var, let, const). Strings: Un "string" es un tipo de dato que conciste en una sucesión de caracteres. Este se define entre comillas ya sean simples o dobles. Ejemplo: "Esto es un string" Funciones (argumentos, return): Una "Función" es una herramienta que nos permite crearla definiendole un nombre, parámetros, y un algoritmo que ejecutará cada vez que la invoquemos. Los argumentos son valores o información que le entregamos a esta función que creamos ya que le serán necesarios como dato para resolver el algoritmo que tiene programado. "return" nos permite decirle a la función que nos devuelva cierto Dato. Por ejemplo: return(x); Esto nos retornará el valor de la variable "x". Hay que tener en cuenta que cuando se va ejecutando el algoritmo y se encuentra con "return", se corta el desarrollo del algoritmo y nos devuelve lo que diga return. La sintaxis de una función es: function nombre_de_la_funcion (argumento_1, argumento_2, argumento_etc){ acá se coloca el algoritmo que queremos que ejecute ... } Declaraciones "if": "if" Es un elemento que nos permite controlar el flujo de nuestro codigo. Necesita que le coloquemos una condición para que si esta condición es verdadera (true), entonces ejecutará el codigo que le coloquemos en su interior y de lo contrario no lo ejecutará y lo pasara por alto. Por ejemplo: if (5 != 4){ console.log("Funciona!") /* Esto nos entregaría por consola "Funciona!", ya que como la condición 5 != 4, es verdadera, entonces el codigo del interior si se ejecuta */ } Valores booleanos ("true", "false"): Los valores booleanos son un tipo de Dato, y existen 2 valores posibles "true" o "false", este tipo de dato es necesario para trabajar con condicionales.
